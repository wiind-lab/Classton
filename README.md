# 클래스톤 (Classton)

교사를 위한 학급 자리 배치 및 학생 관리 도구

---

## 소개

클래스톤은 교사가 학급 운영 과정에서 학생 명단 관리와 자리 배치를 보다 쉽고 효율적으로 수행할 수 있도록 제작된 웹 기반 도구입니다.

설치가 필요 없으며 브라우저에서 바로 사용할 수 있습니다.

---

## 주요 기능

### 학생 관리

* 학생 명단 등록
* 학생 순서 변경
* 다중 학급 관리

### 자리 배치

* 랜덤 자리 배치
* 고정석 지정
* 금지쌍 지정
* 역할 배정

### 데이터 저장

* 브라우저 자동 저장
* 자동 불러오기
* 별도 회원가입 없음

### 사용자 지원

* 내장 사용 안내
* 개인정보 안내

### 이스터에그

* 클래스톤 로고 클릭 시 감성 문구 제공
* 하루 1회 확인 가능

---

## 개인정보 정책

클래스톤은 사용자의 개인정보를 수집하지 않습니다.

학생 명단 및 설정 정보는 사용자의 브라우저(Local Storage)에만 저장됩니다.

인터넷 상의 서버나 외부 저장소에 업로드되지 않습니다.

따라서:

* 다른 사람이 볼 수 없습니다.
* 개발자가 확인할 수 없습니다.
* 다른 컴퓨터에서는 불러올 수 없습니다.

---

## 사용 방법

### 학생 등록

입력 예시

홍길동
김철수
이영희

또는

엑셀, 한글 표의 이름 열을 선택 후 복사하여 붙여넣기

---

### 자리 배치

1. 학생 명단 등록
2. 교실 형태 선택
3. 자리 배치 실행
4. 필요 시 고정석, 금지쌍, 역할 설정
