# Classton

스마트 학급 운영툴

---

## 프로젝트 소개

Classton은 초등 교실에서 사용하는 학급 운영 보조 웹프로그램입니다.

주요 기능

* 자리 배치
* 모둠 관리
* 학생 뽑기
* 다했어요
* 타이머
* 1인1역
* 기록 관리
