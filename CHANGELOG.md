# Changelog

All notable changes to this project will be documented in this file.

## [v1.0.0] - Initial Release

### Added

* 학생 명단 등록 기능
* 다중 학급 생성 및 관리 기능
* 학생 순서 드래그 앤 드롭 변경 기능
* 랜덤 자리 배치 기능
* 자리 배치 결과 저장 및 불러오기 기능
* 고정석 지정 기능
* 금지쌍 지정 기능
* 역할 배정 기능
* 교사용 사용 안내 메뉴
* 브라우저 로컬 저장 기능
* 클래스톤 로고 기반 이스터에그 기능
* 감성 문구 순환 시스템
* 일일 1회 확인 제한 기능

### Improved

* 학생 번호 직접 수정 방식 제거
* 드래그 기반 명단 순서 변경 방식 도입
* 사용 안내 문구 개선
* 개인정보 안내 강화
* 명단 입력창 안내 문구 개선

### Fixed

* 학생 순서 변경 시 고정석 데이터 재매핑 문제 해결
* 학생 순서 변경 시 금지쌍 데이터 재매핑 문제 해결
* 이스터에그 로고 클릭 이벤트 차단 문제 해결
* 팝업 표시 관련 UI 안정성 개선

### Security

* 모든 데이터는 사용자의 브라우저(Local Storage)에만 저장
* 외부 서버 저장 없음
* 개인정보 수집 없음
* 로그인 기능 없음
